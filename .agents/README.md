# 🤖 Agent Workflows

This directory contains workflows and instructions for the AI Agent to maintain continuity across sessions.

## 🛠 Available Commands

### `/resume-work`
- **Location**: [.agents/workflows/resume-work.md](file:///Users/parinkanthakamala/Documents/workspace/transfeero/.agents/workflows/resume-work.md)
- **Purpose**: Restores the agent's memory, syncs the latest code, and prepares the environment for development.
- **When to use**: At the start of a new session.

### `/complete-work`
- **Location**: [.agents/workflows/complete-work.md](file:///Users/parinkanthakamala/Documents/workspace/transfeero/.agents/workflows/complete-work.md)
- **Purpose**: Summarizes the current progress, identifies pending issues, and pushes the state to the repository.
- **When to use**: Before ending a session.

*Note: These are not built-in system commands but instructions for the agent to follow when these "slash commands" are mentioned in the chat.*
