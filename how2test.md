# Transfeero Autopilot Testing Guide

คู่มือนี้จะช่วยให้คุณสามารถทดสอบแอปพลิเคชัน Transfeero พร้อมระบบ Autopilot ได้อย่างถูกต้องและครบถ้วน

## 1. การเตรียมความพร้อม (Prerequisites)
- **Environment**: ตรวจสอบว่าใช้ Java 17 และ Android SDK พร้อมใช้งาน
- **Emulator**: เปิด Emulator ชื่อ `test_device_api34` เตรียมไว้ (หรือใช้คำสั่ง `asimulator`)
- **Login**: คุณต้องมีบัญชี Transfeero Partner เพื่อทดสอบระบบการกดรับงานจริง

## 2. การ Build และติดตั้ง (Build & Install)
ใช้สคริปต์อัตโนมัติที่เตรียมไว้เพื่อความรวดเร็ว:
```bash
./scripts/update_extensions.sh && ./test.sh
```
สคริปต์นี้จะ:

3. ติดตั้ง APK ลงใน Emulator
4. เปิดแอปโดยอัตโนมัติ

## 3. ขั้นตอนการทดสอบ UI & Autopilot
เมื่อแอปเปิดขึ้นมาแล้ว ให้ทำตามขั้นตอนดังนี้:

### ก. การเข้าสู่ระบบ (Login)
1. เข้าสู่ระบบที่หน้าเว็บ **control.transfeero.com** ในตัวแอป
2. ตรวจสอบว่า Cookies (hash/token) ถูกเก็บลงในบราวเซอร์เรียบร้อยแล้ว

### ข. การเปิดใช้งาน Autopilot
1. กดปุ่ม **FAB (Floating Action Button)** สีน้ำเงินที่มุมล่างขวา
2. หน้าต่าง **AUTOPILOT PRO** จะปรากฏขึ้น
3. สังเกตที่มุมซ้ายบน:
    - หากขึ้นสีเขียว **ONLINE**: แสดงว่าดึง Auth จากบราวเซอร์สำเร็จ พร้อมทำงาน
    - หากขึ้นสีเทา **OFFLINE**: ให้กลับไปตรวจสอบว่า Login ในเว็บสำเร็จหรือไม่

### ค. การตั้งค่า Filters & Settings
1. ไปที่แถบ **Filters**: ตั้งค่าราคาขั้นต่ำ, ระยะทาง หรือพิกัดที่ต้องการ (เช่น `airport-milan-50`)
2. ไปที่แถบ **Settings**: ตั้งค่าความเร็วในการรีเฟรช (แนะนำ 1.5 - 3.0 วินาที)
3. กดปุ่ม **Save** ในแต่ละหน้าเพื่อบันทึกค่า

### ง. เริ่มการทำงาน (Running Bot)
1. กดปุ่ม **START BOT** สีเทาด้านล่าง
2. ปุ่มจะเปลี่ยนเป็นสีเขียว **RUNNING**
3. สลับไปที่แถบ **Logs** เพื่อดูการทำงานแบบ Real-time:
    - `Scanning for offers...` (กำลังค้นหางาน)
    - `Found X active offers` (พบงาน)
    - `MATCH: ...` (งานตรงตามเงื่อนไขที่ตั้งไว้)
    - `SUCCESSFULLY ACCEPTED OFFER` (กดรับงานสำเร็จ!)

## 5. กรณีทดสอบพิเศษ (Edge Cases)
- **Rate Limit (429)**: ทดสอบโดยการตั้ง Delay ให้เร็วเกินไป (เช่น 0.5s) บอทควรขึ้นเตือนและหยุดพัก 15 นาทีอัตโนมัติ
- **Daily Limit**: ตรวจสอบว่าเมื่อกดรับงานครบจำนวน (ฟรี 50 งาน) บอทจะหยุดทำงานและแจ้งเตือนให้สมัครสมาชิก
- **Close Window**: กดปุ่มกากบาทมุมขวาบน หน้าต่าง Autopilot ต้องปิดลงโดยที่บอทยังทำงานอยู่เบื้องหลัง (ถ้ากด Start ไว้)
